Wil je Bancontact gebruiken als betaalmethode? Dan moet je aansluiten bij een betaalprovider. Dat kan meestal op een half uurtje na het invullen van enkele vragen over jouw vereniging (in verband met de strenge bankwetgeving, anti-terrorisme en anti-witwassen is dit vereist). Doe dit altijd zorgvuldig.

Je hebt twee mogelijkheden:

- [Aansluiten via Stripe](https://www.stamhoofd.be/docs/stripe/)
- [Aansluiten via Mollie](https://www.stamhoofd.be/docs/mollie/)

Je kan de verschillen in transactiekosten per provider vergelijken via [deze pagina](https://www.stamhoofd.be/docs/transactiekosten/).
