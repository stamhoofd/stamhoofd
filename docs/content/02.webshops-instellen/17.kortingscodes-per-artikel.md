---
title: "Kortingscodes op webshop beperken tot één per besteld artikel"
description: ""
---

Geef je kortingscodes uit, maar wil je niet dat er meerdere codes in dezelfde bestelling gebruikt worden? Dan kan je verhinderen dat er op één artikel meer dan één korting gebruikt kan worden. Je kan niet verhinderen dat er binnen dezelfde bestelling meerdere kortingscodes gebruikt worden, maar wel waarop.

(voor screenshots zie onderaan)

1. Ga naar je webshop in het beheerdersportaal.
2. Onder de instellingen kies je voor 'Kortingen'
3. Maak een nieuwe kortingscode aan (+ kortingscode)
4. Kies en code of genereer een willekeurige code
5. Stel in dat de kortingscode maximaal maar één keer gebruikt kan worden door ‘Beperk aantal keer te gebruiken’ aan te vinken.
6. Klik op '+ korting' toevoegen om een korting die aan deze kortingscode gekoppeld is, toe te voegen.
7. Aangezien we slechts korting willen geven voor één specifiek artikel, kies je voor '+ Artikel'. Stel bovenaan geen algemene kortingen in. Als je bovenaan bv. 5 euro korting zou geven, dan zou je met twee kortingscodes van 5 euro sowieso 10 euro korting krijgen op een bestelling, ook als je maar één artikel besteld.
8. Selecteer bovenaan alle artikels waarop de kortingscode van toepassing is. De korting zal maar op één artikel worden toegepast (zie verder).
9. Kies een kortingsbedrag of percentage. Als je met dezelfde kortingscode korting wilt geven op meerdere artikels, moet je '+ andere korting op tweede stuk' toevoegen.
10. Zorg dat je productkorting op 'niet herhalen' blijft staan. Anders zou de korting op elk artikel toegevoegd worden in het winkelmandje.
11. Vink 'Meerdere kortingen op hetzelfde stuk toestaan' uit. Dat zorgt ervoor dat je niet meerdere keren bijvoorbeeld 5 euro op hetzelfde artikel kan geven - ook niet als je meerdere kortingscodes hebt.
12. Sla op
13. Tip: wil je deze code meerdere keren aanmaken. Klik met je rechtermuisknop op de code en klik op 'Dupliceren' (zie screenshot)
14. Test het eens uit op je webshop! Dan ben je zeker dat het goed werkt zoals je verwacht.
