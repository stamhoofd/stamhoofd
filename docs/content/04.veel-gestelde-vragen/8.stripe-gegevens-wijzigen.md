---
title: "[Stripe] Waar kan ik onze gegevens wijzigen?"
description: ""
---

Via het Stripe dashboard kan je alle gegevens van jouw vereniging wijzigen die gebruikt worden voor de verificatie van je vereniging, en voor de communicatie en beveiliging van Stripe uit. Stripe moet aan allerlei wetgeving voldoen, en heeft daarom allerlei up-to-date informatie van je vereniging nodig.

::note
Werd een document van Stripe afgekeurd? Lees dan [hier](https://www.stamhoofd.be/docs/documenten-stripe-afgekeurd/) welke documenten aanvaard worden ([klik hier](https://www.stamhoofd.be/docs/documenten-stripe-afgekeurd/)).
::

Ga naar Instellingen → Betaalaccounts → Open Stripe Dashboard.

Afhankelijk van de situatie zal je meteen op een pagina terecht komen om je gegevens aan te vullen ([documentatie hiervoor staat hier](https://www.stamhoofd.be/docs/stripe/)). Soms kom je gewoon op het normale Stripe Express dashboard uit. In dit geval moet je zelf naar de instellingen gaan waar je gegevens kan aanvullen door de volgende stappen te volgen.

![](/images/2026/06/s1.png){width="1110" height="1029"}

*Stripe dashboard. Klik hier bovenaan op het persoon icoontje.*

![](/images/2026/06/s2.png){width="1266" height="1221"}

*Hier onderaan bij 'Platforminstellingen' klik je op Stamhoofd*

![](/images/2026/06/s3.png){width="1299" height="1178"}

*In het menu dat rechts uitklapt kan je gegevens aanvullen en corrigeren.*

Hier kan je vervolgens alle gegevens van de vereniging (bedrijfsgegevens) en bestuurders wijzigen.
