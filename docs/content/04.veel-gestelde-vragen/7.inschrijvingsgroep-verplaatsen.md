---
title: "Hoe verplaats ik een inschrijvingsgroep naar een andere categorie of werkjaar?"
description: ""
---

Je kan een groep verplaatsen door met je rechtermuisknop te drukken op de groep &gt; `Verplaatsen naar` . In de app kan je hetzelfde bereiken door de groep lang ingedrukt te houden.

::warning
Je kan hetzelfde bereiken via de instellingen &gt; ledenadministratie &gt; Inschrijvinsgroepen beheren. Hier kan je via de (...)-knop ook groepen verplaatsen.
::

![](/images/2026/06/Screenshot-2026-06-19-at-15.10.48.png){width="735" height="609"}

Als je meerdere werkjaren hebt ingesteld, kan je op dezelfde manier een inschrijvingsgroep verplaatsen naar een ander werkjaar.
